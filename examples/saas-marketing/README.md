# Example — SaaS Marketing Research

A complete lafbot run synthesized from 63 YouTube videos on B2B SaaS marketing.

## How to view

Open `Deep-Research-Report.html` in any browser. It's self-contained — works offline.

## What you'll see

- **TLDR** with 7 most-surprising findings
- **Frequency bar chart** of the top 20 tactics
- **Network graph** showing how tactics cluster and co-occur
- **Anomaly cards** — 24 tactics most founders aren't talking about
- **From the comments** — 18 viewer-sourced insights
- **Contradictions** — 5 places where creators disagree
- **Per-stage playbook** — tactics grouped by Idea → MVP → Beta → V1 → Revenue
- **Source diversity** — top 15 channels contributing to the corpus

## Run parameters

- **Topic**: "B2B SaaS marketing from idea → MVP → beta → V1 → first revenue, with emphasis on 0→1000 waitlist signups, channels beyond outbound/Reddit/Twitter, psychology of launches, anomaly tactics"
- **Date**: 2026-05-24
- **Comment depth**: medium (top 50 per video)
- **Creator scope**: broad mix of indie/bootstrapped, VC-track, sales operators, marketing operators
